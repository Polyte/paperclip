# seeID Marketing Content

Content for the seeID DHA-powered identity verification product. Target audience: South African developers, CTOs, compliance officers, and HR/payroll platform operators.

## Positioning

- **DHA-powered, South Africa-first**: Direct National Population Register access is our moat
- **Lead with speed and authority**: 2-3 second verification, direct DHA integration
- **Compliance-first messaging**: POPIA, FICA — the regulations create demand
- **Fingerprint is a feature, not the headline**: Face-only verification is commoditized
